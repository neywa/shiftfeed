1 What you're testing

ShiftFeed is an OpenShift and Kubernetes news aggregator for SREs, DevOps engineers, and platform engineers. It pulls articles from Red Hat Blog, Kubernetes Blog, CNCF, HackerNoon, and Hacker News, tracks OCP stable releases, surfaces Red Hat CVEs, and generates an AI-powered daily briefing.

Scope of this round: the mobile app on Android and iOS only. Do not test the web build at neywa.github.io/shiftfeed — that is a separate target and out of scope here.

The test plan is split into Free and Pro features. You can complete a full Free pass in about 30 minutes. Pro testing requires a promotional entitlement — see Section 3 for setup.

For each test case below, mark Pass, Fail, or Skip and add a note if needed. Your results will be exported at the end.

2 Environment setup

Android

  1. Install the Play Internal Testing build via the link the developer sent you.
  2. Minimum supported version: Android 7.0 (API 24). Ideally test on Android 13+ for push notification behaviour.
  3. Note the app version — you'll find it at the very bottom of the Settings screen.
  4. Confirm you have Wi-Fi or cellular data available.

iOS

  1. Accept the TestFlight invite sent by the developer and install via TestFlight.
  2. Minimum supported version: iOS 13.
  3. Note the app version from the TestFlight app or from Settings inside ShiftFeed.
  4. Confirm network access as above.
Web and desktop builds are out of scope for this round. Do not file issues against neywa.github.io/shiftfeed.

3 Pro tier setup

Skip this section if you're only testing Free features.

  1. Open ShiftFeed → tap Settings (gear icon).
  2. Go to Account → Sign in → enter your email → tap "Send sign-in link".
  3. Open the magic-link email on the same device → tap the link → ShiftFeed opens signed in.
  4. Back in Settings, scroll to the very bottom → tap Copy RevenueCat ID.
  5. Send that ID to the developer via email or Slack. They will grant a Pro promotional entitlement.
  6. Fully restart the app. A PRO badge should appear in the top app bar.
If the PRO badge doesn't appear after restarting, tap Settings → scroll to bottom → "Restore purchases" and try again.
